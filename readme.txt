== Description ==

Another Autbox Box is a wordpress plugin that shows your visitors an author box.

It allows you to add a few social media links such as Twitter, Facebook and Linkedin with icons.

It uses gravatar for author images and uses wordpress profile descrtion for box description. 

Lastly some configuration about social links and etc.

== Installation ==

1. Upload `another-author-box.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure it on Settings -> Author Box.
